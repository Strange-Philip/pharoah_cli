## Setup

### Prerequisites
Make sure you have Git installed on your machine before using this tool.

To install:
```shell
pub global activate yaroo_cli
```

To update:
```shell
pub global activate yaroo_cli
```

## Usage

```shell
yaroo_cli help
```

For help on specific command, `run yaroo_cli help [command name]` For example:

```shell
yaroo_cli help create 
```
will show how to use command create 


## Generating Yaroo project

```shell
yaroo_cli create projectName
cd project_name
pub get
```
Navigate to `http://localhost:3000` to visit the project you just built. Command following will assume that you are in the root directory of the project.
